# Escaping the Browser Sandbox Episode 1

## Showing web devs alternatives to web technology.

I'm starting a new series today. I want to show
web developers that the prescribed "use react for everything"
approach has valid alternatives. not everything needs to be electron
or react native.

I want to show this by building a project.

I'm not sure exactly what that project will be yet,
but here's criteria:

1. must have a website, desktop client, and mobile app.
2. website must work across browsers, desktop must work across operating systems, mobile app must work on ios and android.
3. as much code re-use as possible.
